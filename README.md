Ani-ON
=====
https://anion.herokuapp.com

Ani-ON은 [애니시아](https://www.anissia.net)의 [애니편성표 API](https://www.anissia.net/?m=1&b=4)를 이용하여 만든 간단한 웹 어플리케이션입니다.

특징
-----
 - 모바일 웹 브라우저도 지원함.
 - 종영 애니메이션 목록도 볼 수 있음.
 - 검색 기능 및 장르별 보기 제공함.

라이센스
-----
Ani-ON의 소스코드는 MIT 라이센스를 따릅니다.
다만, 애니시아의 API사용에 대해선 애니시아측에 문의해주시기 바랍니다.
